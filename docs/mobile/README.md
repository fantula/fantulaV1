## ⚠️ 核心原则 (必读)

```
┌─────────────────────────────────────────────────────────┐
│                    移动端开发原则                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ✅ 只开发 UI 布局，不参与任何业务逻辑                   │
│                                                         │
│  ✅ 数据获取逻辑 完全复用 PC 端实现                      │
│                                                         │
│  ✅ 使用现有 API，不新增、不修改接口                     │
│                                                         │
│  ✅ 状态管理、计算逻辑 与 PC 端保持一致                  │
│                                                         │
│  ❌ 不复制一套业务逻辑                                   │
│                                                         │
│  ❌ 不为移动端单独实现数据请求                           │
│                                                         │
│  ❌ 不引入新的业务状态                                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 🔗 与 PC 端的关系

```
┌─────────────────────────────────────────────────────────┐
│                    共享层 (禁止复制)                     │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  api/client/          → PC/Mobile 共用 API              │
│  stores/              → PC/Mobile 共用状态              │
│  composables/         → PC/Mobile 共用业务逻辑          │
│                                                         │
│  ⚠️ 移动端必须 import 这些，绝不能复制或重写            │
│                                                         │
└─────────────────────────────────────────────────────────┘
                          │
            ┌─────────────┴─────────────┐
            │                           │
            ▼                           ▼
┌───────────────────────┐   ┌───────────────────────┐
│        PC 端          │   │       移动端          │
├───────────────────────┤   ├───────────────────────┤
│  pages/               │   │  pages/mobile/        │
│  components/pc/       │   │  components/mobile/   │
│  layouts/default.vue  │   │  layouts/mobile.vue   │
│                       │   │                       │
│  桌面布局             │   │  触屏布局             │
│  鼠标交互             │   │  手势交互             │
│  悬停效果             │   │  底部导航             │
└───────────────────────┘   └───────────────────────┘

         逻辑必须完全一致，只是 UI 不同
```

## ❌ 禁止事项

1. **禁止新增 API 接口**
2. **禁止复制 PC 端业务逻辑**
3. **禁止创建新的数据状态**
4. **禁止改变操作流程**
5. **禁止使用 Element Plus** (移动端应轻量化)

---

## ✅ 允许的操作

1. **优化布局** - 适配移动端屏幕
2. **优化交互** - 触摸手势、滑动效果
3. **添加移动端专属组件** - 底部导航、抽屉等
4. **优化加载体验** - 骨架屏、下拉刷新
