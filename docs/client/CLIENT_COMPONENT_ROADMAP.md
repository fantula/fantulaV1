## 🔧 扫描检查项

每个页面扫描时，检查以下内容：

### 全局化检查

```markdown
### 弹窗检查
- [ ] 是否使用 BaseModal?
- [ ] 如用自定义，样式是否与 BaseModal 一致?
- [ ] 是否需要替换为 BaseModal?

### 按钮检查
- [ ] 是否使用 BaseButton?
- [ ] 如用自定义，样式是否与 BaseButton 一致?
- [ ] 是否需要替换为 BaseButton?

### 卡片检查
- [ ] 是否使用全局卡片组件?
- [ ] 是否有重复的卡片样式?
```

### 清理检查

```markdown
### 代码清理检查
- [ ] 有无 console.log?
- [ ] 有无未使用的变量?
- [ ] 有无未使用的函数?
- [ ] 有无重复代码?
- [ ] 有无硬编码样式?
- [ ] 有无硬编码文案?
```
